# go-chat

#### An online chat room with go and vue.

---

#### 功能列表:

1. 用户管理

   - 🔨 用户注册、登录（JWT 鉴权）
   - 头像、昵称修改
   - 好友管理（加好友、删除好友）

2. 即时通讯

   - 单聊（P2P 聊天）
   - 群聊（支持群成员管理）
   - 在线状态（用户在线/离线显示）

3. 消息类型

   - 文字、图片、表情、文件发送
   - 消息已读/未读状态
   - 消息撤回

4. 离线存储

   - 离线消息存 Redis / MySQL / MongoDB
   - 消息队列（Kafka）
   - 保证可靠消息投递

5. WebSocket 连接管理

   - 断线重连
   - 负载均衡（Nginx / Gateway）

6. 前端

   - Vue.js 组件化 UI
   - 消息列表、好友列表
   - 输入框支持表情 & 文件上传
